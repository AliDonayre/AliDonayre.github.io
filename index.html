Sobre mi ...

Arquitecta de Interiores, artista en sus tiempos libres pues amo mucho dibujar y pintar. Madre de dos felinos , tratando de mantenerlos trabajando y estudiando a la vez.

Actualización: Aun sobrviviendo

Semana 1

En esta primera semana pudimos conocernos entre nuestros compañeras de una manera mas creativa, en la cual nos dibujamos y describimos como frutas, fue una manera muy curiosa para conocernos y de esta manera formar los grupos. Empezamos tambien el desarrollo de nuestro blog , para usar el html y escoger una plantilla que mas nos guste para mostrar.

31 de Agosto del 2023

Semana 2

En esta segunda semana de clase ya teniamos grupos formados y realizamos una exposición sobre un producto en especifico, con mi grupo escogimos la Inca Kola. Tambien en esta semana empezamos a utilizar el lenguaje de programación html para relaizar esta pagina web y usarla a forma de bitacora semanal.

Semana 3

Esta semana escogimos el proyecto que vamos a realizar en este curso, escogimos el Proyecto Capsule. En base al proyecto escogido realizamos un guión de una anecdota que se pudiera relacionar con el producto, fue muy dvertido escoger las imagenes para acompañar la anécdota.
